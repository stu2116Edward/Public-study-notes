### 提取文件内容
保证整行里同时出现顺序不限  
按下`Ctrl+F`在查找中输入正则表达式内容如下，下方**选择正则表达式**，点击`提取`最后保存  
```
^(?=.*时间2003)(?=.*地点)(?=.*人物).*$
```

<img width="899" height="524" alt="屏幕截图 2025-12-06 232132" src="https://github.com/user-attachments/assets/722fda0f-3d95-4ce3-9d12-a50a9586a07c" />  

如果要新增在`^`和`).*$`中间增加`(?=.*关键字)`  
测试正则表达式：https://regex101.com  
在 EmEditor（PCRE 引擎）里，只要在“关键字”里遇到下面这 14 个字符，就必须先转义，否则正则会被拆错：  
<pre>
\     ^     $     .     |     ?     *     +     (  )     [  ]     {  }
</pre>
记忆口诀:  
“**斜杠尖点杠，竖问星加圈，括号配三方，花号也要转**。”  

其它注意:  

1. 空格、汉字、数字、下划线、逗号、冒号、引号、横线 `-` 都不需要转义  
2. 如果关键字里本身就有 `\`，要先写成 `\\`  
3. 想匹配“整段数字”可用 `\d+`；匹配“空白”用 `\s`；匹配“非空任意字符”用 .（此时别忘了勾选“`.` 匹配换行符”选项，或在表达式开头写 `(?s)`  
4. 大小写敏感：默认区分。若要不区分，在查找面板勾“忽略大小写”或在正则开头加 `(?i)`  
5. 多行模式：如果关键字可能跨行，用 `(?s)` 让 `.` 匹配换行符；用 `(?m)` 让 `^` `$` 匹配行首行尾而非整个文本首尾



### 使用 Emeditor 删除文件中的空行
使用快捷键`Ctrl + H`替换  
勾选使用**正则表达式**  
查找中输入：
```
^[\s]*\n
```
或者
```
^[\t]*\n
```
- `^`：行首断言
- `[\s]`：`\s` 已包含所有空白字符（空格、TAB、回车、换页、VT 等）
- `[ \t]`：仅匹配空格（ASCII 32）或制表符（TAB，ASCII 9）
- `*`：前述字符出现 0 次或多次
- `\n`：换行符（LF，ASCII 10）

`^[\t]*\n`　→　只认**空格/TAB**构成的空白行  
`^[\s]*\n`　→　认**所有空白字符**构成的空白行

替换为**空**  
点击**替换全部**
